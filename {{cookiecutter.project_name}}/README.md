# Проект для тестирования {{ cookiecutter.project_slug }}

### Стек

* Язык программирования: **Python**


### Структура проекта

```
 ├── o3                       # Папка с настройкой CI
 ├── app                      # Папка с базовыми специфичными для проекта проверочными и вспомогательными функциями
 │   ├── clients              # Папка с сгенерированными клиентами 
 │   └── fixtures             # Папка с сгенерированными фикстурами 
 │       └── config.py        # Файл c конфигурацией проекта (VAULT_PATH , O3_SERVICE_NAME)
 ├── tests                    # Папка для хранения тестов
 │   ├── smoke                # Чтобы был запуск в пайплайне тесты должны располагаться в папка smoke/functional
 │   └── conftest.py          # conftest с импортированными, сгенерированными фикстурами
 ├── pb                       # Папка где располагаются сгенерированные gRPC стабы
 ├── .gitignore               # Стандартный файл для игнорирования нежелательных файлов для загрузки в репозиторий
 ├── gitlab-ci.yml            # Файл с подключением проекта к пайплайну для проектов с тестами
 ├── nuke.toml                # Файл конфигурации проекта, необходимо заполнить для генерации клиентов и фикстур
 └── README.md                # Файл с описанием репозитория для тестов
```
